- Couple of weeks back, I started using Copilot CLI in autopilot mode, and it was better than IDE
  1. Terminal UI is better, it shows how agent is solving a problem, as IDE hides that information
  2. IDE used to run into UI errors, connectivity errors -> Sometimes forcing to start a new chat session
  3. CLI seems to have better performance, maybe because IDE handles a lot of load in rendering the message
  4. Better exposure of commands
  5. Feels more programmatic and agentic engineering while IDE tends to feel like vibe coding
